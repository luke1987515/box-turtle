# Windows 多磁碟壓力測試工具 (FIO Automation Script)

這是一個基於 PowerShell 的自動化腳本，專為伺服器 DQA 測試、JBOD 驗證或大規模 HDD 壓力測試環境設計。它能自動偵測系統中所有非 OS 的實體磁碟，並透過 `fio` 進行高併發的寫入壓力測試。

## 📌 功能亮點
- **安全過濾**：自動識別並排除系統碟（IsSystem），防止誤刪除作業系統資料。
- **解決長度限制**：自動生成 `.fio` 設定檔，避開 Windows 指令行 260 字元的長度限制。
- **消除冗餘警告**：內建 `thread` 參數，消除 Windows 平台上的 Process Shared Mutex 警告。
- **彙整報表**：使用 `group_reporting` 提供所有磁碟加總的頻寬 (Throughput) 與 IOPS 數據。

## 🛠️ 前置作業
1. **安裝 fio**: 
   請確保已安裝 `fio` 並加入系統路徑。
   ```powershell
   winget install fio
   ```
管理員權限:
此腳本需要存取實體磁碟路徑 \\.\PhysicalDrive，必須以 系統管理員身份 執行 PowerShell。

## 🚀 使用說明
1. 下載並儲存腳本為 StressTest.ps1。

2. 開啟管理員權限的 PowerShell 視窗。

3. 執行腳本：

PowerShell
```
.\StressTest.ps1
```

4. 根據畫面顯示的磁碟清單核對資訊。

5. 輸入測試參數（或直接按 Enter 使用預設值：60秒, 1M Block Size）。

6. 輸入 Y 並按 Enter 開始執行。

## 📊 結果觀察重點
測試結束後，重點查看 Run status group 0 區段：

WRITE: bw=...: 顯示總體吞吐量（例如：bw=7500MiB/s）。

IOPS=...: 顯示總體每秒輸入輸出次數。

Lateny (clat): 觀察平均與百分位延遲，確認磁碟反應是否一致。
